## 代码的编译运行

### 代码的下载和包安装

```bash
$ git clone https://gitee.com/zvms/zvms-electron.git
$ cd zvms-electron
$ npm install
```

### 热加载
```bash
$ npm run electron:serve
```

### 编译
```bash
$ npm run electron:build
```
